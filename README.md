# HH Bot Dashboard

Веб-интерфейс для автоматизации работы с hh.ru: массовые отклики, авто-подъём резюме, AI-ответы работодателям, заполнение опросников нейронкой, детекция роботов-рекрутеров.

## Возможности

- **Массовые отклики** — параллельный сбор вакансий по нескольким поисковым URL, фильтрация, автоматическая отправка откликов
- **LLM авто-ответы** — DeepSeek/OpenAI отвечает на сообщения HR, учитывает резюме и историю переписки
- **Заполнение опросников** — нейронка заполняет анкеты работодателей при откликах
- **Детекция роботов-рекрутеров** — распознаёт чат-ботов HR, отвечает кнопками вместо текста
- **OAuth API** — отклики через официальный API hh.ru (опционально)
- **Умные фильтры** — по зарплате, формату работы, исключение агентств
- **Авто-подъём резюме** — поднимает резюме в поиске каждые 4 часа
- **Дневной лимит** — контроль количества откликов в день с автосбросом
- **Мониторинг** — просмотры резюме, приглашения, статистика переговоров
- **Мультиаккаунт** — несколько аккаунтов параллельно (браузерные сессии)
- **WebSocket** — обновление дашборда в реальном времени (300мс)

## Быстрый старт

### Docker (рекомендуется)

```bash
git clone <repo>
cd hh.ru-clicker
docker-compose up -d
```

Открыть: http://localhost:8000

### Локально (Python)

```bash
git clone <repo>
cd hh.ru-clicker
pip install -r requirements.txt
python web_app.py
```

Открыть: http://localhost:8000

## Структура проекта

```
hh.ru-clicker/
  web_app.py              # FastAPI app, BotManager, routes
  app/
    config.py             # Config, load/save настроек
    storage.py            # Кэш: applied, tests, interviews
    state.py              # AccountState — состояние аккаунта
    hh_api.py             # Парсинг страниц HH
    hh_apply.py           # Отправка откликов, опросники
    hh_chat.py            # Chatik API — переписка
    hh_resume.py          # Парсинг и аудит резюме
    hh_negotiations.py    # Статистика переговоров
    llm.py                # LLM интеграция (DeepSeek/OpenAI)
    questionnaire.py      # Парсинг и заполнение опросников
    oauth.py              # OAuth2 токены HH
    websocket.py          # WebSocket менеджер
    logging_utils.py      # Логирование
  static/
    index.html            # HTML структура дашборда
    css/style.css         # Стили
    js/app.js             # JS логика
  data/                   # Данные (не в git)
    config.json           # Настройки бота
    browser_sessions.json # Аккаунты и cookies
    applied_vacancies.json# История откликов
    test_vacancies.json   # Вакансии с тестами
    interviews.json       # Переговоры
```

## Данные

Все данные хранятся в `data/` (монтируется как Docker volume):

| Файл | Описание |
|------|----------|
| `config.json` | Настройки: URL пул, лимиты, LLM конфиг |
| `browser_sessions.json` | Аккаунты: имя, cookies, resume_hash |
| `applied_vacancies.json` | История откликов (по аккаунтам) |
| `test_vacancies.json` | Вакансии с опросниками |
| `interviews.json` | Переговоры и чаты |

## Технологии

| Компонент | Стек |
|-----------|------|
| Backend | Python 3.11, FastAPI, uvicorn |
| Frontend | Vanilla JS, WebSocket (real-time 300ms) |
| AI | DeepSeek / OpenAI совместимый API |
| Парсинг | requests, aiohttp, SSR JSON, regex |
| Чат | chatik.hh.ru API (reverse-engineered) |
| OAuth | HH Android app credentials |
| Деплой | Docker + docker-compose |

---

## Требования

- Python 3.10+
- Cookies активной сессии hh.ru (hhtoken, hhuid, _xsrf, crypted_id)
- API ключ DeepSeek/OpenAI (для LLM-функций, опционально)
